# 抖音小红书下载器 (DyXhsDL)

一个 Android 应用，支持 **抖音** 与 **小红书** 无水印内容下载。抖音走直链去水印下载，小红书支持图片、视频、文案与 Live Photo 提取。

> 小红书 iOS 版本仓库 ➡️ [点我](https://github.com/NEORUAA/XHS_Downloader_iOS)

## 主要功能

### 抖音
- **去水印直链下载**：解析分享链接后直接拉取无水印视频（`playwm` → `play`），无需借助 WebView 中转。
- **规避风控**：解析与下载均使用随机 iPhone UA（Safari / CriOS / EdgiOS / FxiOS 轮换），降低被抖音 security 风控拦截的概率。

### 小红书
- **图片下载**：自动提取笔记中的图片并获取其原始 CDN 地址。
- **视频下载**：自动提取笔记中的视频（至多 1080p）。
- **文案提取**：自动提取笔记文案并复制到剪切板。
- **Live Photo 合成**：智能识别并合成动态照片，将图片和视频组合成一个动态照片文件（支持小米、OPPO、谷歌的动态照片格式）。
- **网页版提取**：针对不支持 okhttp 方式获取内容的笔记，可选从内置 WebView 爬取内容。

### 通用
- **前台服务下载**：切后台或锁屏时下载不中断。
- **剪切板自动识别**：复制分享链接后自动检测平台（抖音 / 小红书）并提示下载，支持去重避免重复触发。
- **历史任务**：下载记录带平台徽标（抖音青 / 小红书红），可重试、续传、停止、删除。

## 使用说明

### 基本操作
1. **复制链接**：在抖音或小红书 App 中复制分享链接（抖音支持 `v.douyin.com` 短链与 `douyin.com/video/...` 长链；小红书支持 `xiaohongshu.com` 与 `xhslink.com`）。
2. **自动/手动**：应用检测到剪贴板链接会自动提示，也可在输入框手动粘贴。
3. **开始下载**：点击下载，等待处理完成。
4. **查看内容**：下载完成后在系统「下载 / `xhsdn`」目录（`Download/xhsdn`）中查看。

### 支持的链接类型
- 抖音：`https://v.douyin.com/...`、`https://www.douyin.com/video/...`、`https://www.iesdouyin.com/share/video/...`
- 小红书：`https://www.xiaohongshu.com/explore/...`、`https://xhslink.com/...`、`https://www.xiaohongshu.com/discovery/item/...`

## 常见问题
1. **抖音下载失败**：多为抖音风控拦截。可重试（每次会使用新的随机 UA）；若持续失败，可尝试先用浏览器打开分享链接再复制。
2. **小红书下载失败**：请确认分享链接在浏览器中可被查看。
3. **Live Photo 无效**：合成失败会自动 fallback 成普通视频下载；部分设备可能不支持或需要特定应用才能播放。
4. **找不到文件**：文件保存在系统「下载」目录下的 `xhsdn` 文件夹（`Download/xhsdn`），请在文件管理器或相册中查看。
